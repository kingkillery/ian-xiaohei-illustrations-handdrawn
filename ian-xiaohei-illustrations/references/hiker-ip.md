# 徒步 IP（Hiker / Xiaohei Replacement）

## 角色定义

徒步是 Ian 中文正文配图的新版固定视觉 IP，替代原来的小黑。徒步每张图都出现，承担核心动作（搬运 / 卡住 / 操作 / 盖戳 / 拉线 / 捞东西 / 切分 / 铺路 / 压面 / 发酵 / 盖章 等），不能只是装饰。

徒步不是吉祥物，不是可爱卡通，而是穿着户外装备、认真做一件荒诞但成立的工作的人。

## 外形

- 男生，轻度手绘风格，线条略抖、不矢量、不厚重描边。
- 浅棕色短发、修剪整齐的棕色短胡须。
- 蓝色棒球帽（帽檐朝前）。
- 蓝色头戴式耳机（戴在帽子外侧，包住耳朵）。
- 灰色连帽卫衣，胸前有一个小三角 logo（黑色细线）。
- 蓝色牛仔裤。
- 普通运动鞋 / 徒步鞋。
- 表情：冷静、空白、认真、deadpan、不笑、不卖萌。
- 站立时直立、行进时身体略前倾、蹲下时膝盖弯曲。
- 身体比例偏瘦长，动作有点笨拙但不蠢。

## 性格

- 认真做事但事有点荒诞。
- 像一个低调的户外操作员。
- 冷幽默，不卖萌。
- 有点笨拙但不蠢。
- 像在白板草图里真的负责某个工作。

## 常见职责（与小黑对齐）

让徒步承担核心动作：

- 搬运素材。
- 拉线汇聚信息源。
- 卡在断点里。
- 在机器里操作"判断"杆。
- 变成筛选漏斗。
- 切开"素材鱼"。
- 盖章承接话术。
- 牵着承接路径。
- 举警告牌看坑。
- 从洞里伸手但接不住内容。
- 在旁边搬砖、搭桥、开门、分拣、记录。

## 禁止

- 不要把徒步画成过度可爱的吉祥物。
- 不要画成儿童卡通 / Q 版角色。
- 不要写实摄影、复杂阴影、写实服装褶皱。
- 不要让徒步只是站在角落里看。
- 不要让徒步抢走结构表达（结构优先于角色）。
- 不要让徒步穿上非徒步场景的奇装异服（保留帽子、耳机、卫衣、牛仔裤、徒步鞋）。

## 判断标准

如果去掉徒步，图的核心隐喻还能完全成立，说明徒步太装饰了；要重写提示词，让徒步成为动作主体。

## 颜色使用（继承 DNA）

徒步本人只在黑色 / 灰色 / 蓝色出现，不引入其他颜色。帽、耳机、牛仔裤 = 蓝；卫衣 = 灰（线稿 + 淡灰填充）；头发 / 胡须 = 棕（用细黑线 + 极淡的暖灰线表示，不破坏极简感）；鞋 = 黑 / 深灰。其他物体和标注沿用 红 / 橙 / 蓝 三色克制用法。
